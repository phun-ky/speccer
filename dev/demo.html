@phun-ky/speccer

A zero dependency package to highlight elements

Speccer adds specification to your HTML-elements

Project page Code on github README 
$ npm i @phun-ky/speccer --save

All examples are activated with:

import speccer from "https://esm.sh/@phun-ky/speccer";
…
speccer();
Note

The animations are just sugaring, a way for the examples to come alive.

Mark elements

<div
  data-speccer="mark"
  class="box …"
>
  …
</div>

Accessibility

<button
  type="button"
  data-speccer="a11y shortcut"
  data-speccer-a11y-shortcut="ctrl + s"
  class="save …"
>
  …
</button>
